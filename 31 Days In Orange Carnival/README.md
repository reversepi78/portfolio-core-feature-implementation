# 31 Days In Orange Carnival

## 소개
31 Days In Orange Carnival은 비주얼 노벨 기반의 로그라이크와 인터랙티브 픽션 요소가 결합된 게임입니다.
플레이어의 선택에 따라 스토리 흐름이 변화하고, 다양한 엔딩으로 이어집니다.
로그라이크 구조를 적용해 매 플레이마다 다른 선택과 루트, 엔딩을 경험할 수 있도록 구성했습니다.
