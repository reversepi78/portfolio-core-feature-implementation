# 플레이어 상태 기반 동적 이벤트 확률 튜닝 시스템

## 기능 개요
플레이어 상태를 정규화된 지표로 변환하고, 이를 기반으로 이벤트 발생 확률을 동적으로 조정하는 시스템

## 핵심 흐름
1. 초기 기본 확률 로드
2. DistressCalculator 플레이어 현재 상태 정규화
3. 정규화 값에 따라 ScriptCategoryRoulette에서 이벤트 카테고리들의 확률 튜닝
4. 히스토리 데이터를 참조하여 연속으로 같은 카테고리의 이벤트가 발생했을 경우 확률 튜닝

## 주요 코드
DistressCalculator.cs → 플레이어의 현재 상태를 정규화함

ScriptCategoryRoulette.cs → 플레이어의 현재 상태에 따라서 이벤트 카테고리들의 확률을 튜닝함
