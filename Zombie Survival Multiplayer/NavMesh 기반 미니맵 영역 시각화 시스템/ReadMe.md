# NavMesh 기반 미니맵 영역 시각화 시스템

## 기능 개요
NavMesh 데이터를 활용해 이동 가능한 영역을 자동으로 Mesh로 생성하고, 미니맵에 시각적으로 표시하는 시스템

## 핵심 흐름
1. NavMesh 데이터를 기반으로 미니맵에 표시할 맵 Mesh 생성
2. 플레이어 상단을 따라다니는 미니맵 전용 카메라를 구성하고, 렌더링 결과를 Target Texture에 출력
3. 플레이어, 좀비, 아이템에 미니맵용 아이콘 오브젝트를 할당하고, 전용 Layer를 설정하여 미니맵 카메라에서만 렌더링되도록 처리

## 주요 코드
MiniMapNavMeshMeshBuilder.cs → 구워진 NavMesh를 Mesh로 제작하여 미니맵 카메라에서 보일 수 있게 만들어줌

ObjectSpawner.cs → 좀비나 아이템을 생성할 때 미니맵에 표시할 아이콘도 함께 생성
