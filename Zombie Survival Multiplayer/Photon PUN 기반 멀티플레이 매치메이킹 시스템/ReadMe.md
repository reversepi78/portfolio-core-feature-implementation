# Photon PUN 기반 멀티플레이 매치메이킹 시스템

## 기능 개요
Photon PUN 기반 로비 매치메이킹 및 Host 권한 중심 멀티플레이 시스템

## 핵심 흐름
1. 로비에서 매치메이킹을 통해 룸을 생성하거나 참가하고, 함께 플레이할 인원과 게임에 입장
2. 마스터 클라이언트에서 주요 게임 데이터를 처리
3. 다른 클라이언트는 마스터 클라이언트에서 전달된 데이터를 기반으로 상태를 동기화

## 주요 코드
- LobbyManager.cs → 로비 씬에서 룸 생성 및 참가 등 매치메이킹 처리
- 
- GameManager.cs → 메인 게임 씬에서 마스터 클라이언트 기반 리슨 서버 구조 구현
