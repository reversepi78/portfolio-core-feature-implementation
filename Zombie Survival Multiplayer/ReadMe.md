# Zombie Survival Multiplayer

## 소개
Zombie Survival Multiplayer은 Photon PUN을 기반으로 구현한 멀티플레이 슈팅 게임입니다.
탑다운 시점과 FPS 시점을 자유롭게 전환하며 플레이할 수 있도록 구성했으며,
각 게임 방에서는 최대 4인까지 함께 플레이를 지원합니다.
