# Cinemachine 기반 시점 전환 시스템

## 기능 개요
Cinemachine 가상 카메라를 활용하여 탑다운과 FPS 시점을 자유롭게 전환할 수 있도록 구현한 시점 전환 시스템

## 핵심 흐름
1. 탑다운 시점으로 게임 시작
2. 탑다운 시점의 가상 카메라는 플레이어와 일정 거리 떨어진 탑뷰 위치에서 플레이어를 추적하고, FPS 카메라는 플레이어 캐릭터의 눈높이에서 플레이어의 회전과 동일하게 움직인다.
3. 이후 v키 입력을 통해 탑다운-FPS 시점을 오가며 게임 진행

## 주요 코드
CameraModeManager.cs → 카메라 시점을 전환하고, 각 시점에 맞는 설정을 적용
